# BD-A-Aplica-o-de-Gerenciamento-de-Pedidos

Plataforma de e-commerce nos moldes da Amazon. Os usuários precisam
conseguir:
• Cadastrar clientes, vendedores e itens;
Plácido Neto — IFRN / TADSPágina 1
IFRN — Instituto Federal do Rio Grande do Norte Introdução ao Django Rest Framework
• Visualizar os itens disponíveis;
• Adicionar itens ao carrinho e definir um endereço de entrega;
• Realizar um pedido e escolher a forma de pagamento;
• Consultar pedidos realizados, itens comprados e itens vendidos.
Toda essa lógica será exposta por meio de endpoints REST que retornam e recebem dados no
formato JSON, permitindo que qualquer cliente — um aplicativo mobile, um front-end
React/Angular ou um script Python — interaja com o sistema
